# Quick-Calc

Quick-Calc is a simple calculator project for the Software Testing & Quality Assurance practical submission.
It supports: addition, subtraction, multiplication, division (with division-by-zero handling), and clear.

## Features
- Add, subtract, multiply, divide
- Division by zero raises an error
- Clear resets the current value to 0

## Project structure
- `src/` - calculator logic
- `tests/` - unit and integration tests
